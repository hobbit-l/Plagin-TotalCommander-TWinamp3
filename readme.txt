Это - оригинальные исходные коды плагина TWinAmp3 для Total Commander (http://www.tckb.ru/wiki/TWinAmp).
Выложено на github в надежде, что кто-то продолжит разработку.

В предлагаемые исходные коды включён набор модулей Audio Tools Library (файлы ID3v1.pas, ID3v2.pas, MpegAudio.pas), автором которых является Jurgen Faul. Информацию о этих модулях вы можете найти в них самих.
